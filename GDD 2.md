# 一、 游戏概述与核心卖点（USP）

### 1.1 一句话定义

玩家在动态变化的等距水晶浮岛网格中，通过拖拽随机发放的“水晶路径碎片”构建光路，引导光流在时间压力下完成折射通关。

### 1.2 核心卖点（USP）

- 视觉惊艳（一秒吸睛）：等距低多边形半透明水晶风格（Isometric Crystal Low-Poly），利用高饱和度的冰蓝/粉紫流光在暗色背景上形成强烈反差，具备独立精品游戏的质感。
    
- 即时正反馈（拖拽即看）：采用“拖拽即预演”机制，玩家每移动一步都能看到光流路径的实时模拟，极大地消除了挫败感。
    
- Crystal Flow 多级爽感：当路径连通时，光流瞬间一气呵成爆发，配合完美的音效和震动，提供极强的秩序解压感。
    
- 极短局碎片化：单局 30–90 秒，完美适配微信生态下的无压力碎片化场景。

### 2.2 核心操作

- 拖拽放置：将底部手牌拖入网格内可放置区域。
    
- 点击旋转：部分高级碎片放置后，点击可改变其内部的偏振/折射方向。
    
- 弃牌刷新：丢弃当前手牌以获取新牌（局内限制次数或通过看广告重置）。

# 三、 微信小游戏调优：上瘾与裂变设计

### 3.1 极速上手（FTUE）设计

为了防止极轻度用户在 1 分钟内因为看不懂规则而流失，前 5 关采取“零挫败强引导”：

- 第 1 关（绝对直观）：全屏变暗，仅留 1 个起点、1 个终点和 1 个高亮网格。手牌强制锁定 1 张直线水晶。玩家只要拖上去，无需点击开始，光流自动发射连通，完成第一次视觉震撼。
    
- 第 2-3 关（机制拆解）：加入折角，不引入限时，手牌无随机，强制玩家理解“转弯”概念。
    
- 第 4-5 关（爽感初现）：引入“Perfect Flow（完美路径）”概念，引导玩家通过满星过关。
    

### 3.2 Near Miss（差一点）成瘾机制

这是提高拉回率和广告变现的核心心理机制：

- 触发条件：当时间耗尽或手牌用完时，若路径距离终点仅差 1-2 格，或者只差一个颜色门未解开。
    
- 视听表现：游戏画面瞬间进入慢动作（Bullet Time），镜头拉近切至终点特写，背景音乐音调上升并伴随心跳声，屏幕边缘出现亮光闪烁。
    
- 挽救策略：弹出黄金拯救弹窗——“召唤万能碎片（看广告/群求助）”，给予玩家极强的心理暗示去完成“未竟的强迫症”。
    

### 3.3 微信专属：异步社交与裂变钩子

拒绝无意义的弹窗分享，采用以下天然具备社交属性的机制：

- 钩子一：【残局挑战卡】（微信群扩散）
    

- 机制：玩家遭遇极难关卡失败，或打出超越 95% 人的 Perfect 高分时，可一键生成“残局/炫耀”卡片发到微信群。
    
- 文案示例：“我用 3 个折射镜连击了 99 次完美通关！全群谁能破这个局？”
    
- 收益：群友点击卡片可直接直接免下载进入该特定关卡残局。群友挑战成功，分享者与挑战者均可获得“珍稀光能”。
    

- 钩子二：【互助充能】（裂变体力）
    

- 游戏引入体力值限制（单局消耗 1 点）。体力耗尽时，除看广告外，可发送“呼叫充能”至微信好友，好友点击助攻即可帮玩家回复 1 点体力，好友自身也能获得局外抽奖券，实现双向裂变。
    

# 四、 商业化变现方案（混合变现）

根据微信小游戏的特点，本项目采用 IAA（广告为主，占85%）+ IAP（轻度内购，占15%） 的变现结构。

### 4.1 深度广告（IAA）点位设计

|   |   |   |   |
|---|---|---|---|
|场景模块|广告触发点|玩家核心痛点|刺激度与转化率|
|局内拯救|Near Miss 拯救|差一步通关，看广告直接获得1张可任意旋转的“万能水晶”或延长30秒。|极高 🔥🔥🔥|
|局内洗牌|手牌发牌员制裁|手牌卡死无法破局，看广告免费获得一次“全手牌刷新”机会。|高 🔥🔥|
|结算暴击|完美通关（Perfect）|达成完美解，看广告可将本局获得的“光能/碎片经验”翻3倍。|中 🔥|
|局外挂机|离线光能裂变|重新登录游戏，看广告可一键领取 3 倍的离线挂机光能产出。|高 🔥🔥|

### 4.2 轻度内购（IAP）点位设计

- 视觉定制化（皮肤）：
    

- 光流特效：如“赛博霓虹”、“地狱岩浆”、“金色流光”等高亮尾迹特效。
    
- 水晶主题：改变全关卡水晶低模外观的“黑曜石风格”、“赛博朋克风”。
    

- 便利性礼包：无广告特权月卡（免看广告直接获得拯救/洗牌奖励）、关卡包解锁。
    

# 五、 视觉与技术美术（TA）规范

### 5.1 艺术风格（Art Style）

- 风格定位：Isometric Crystal Low-Poly Style（等距低多边形半透明水晶风格）。
    
- 色彩矩阵：
    

- 底色背景：深蓝紫渐变（#0B091A 到 #1D183A），烘托宇宙深邃感，让光线更扎眼。
    
- 常规路径：高亮冰蓝（#00F0FF），流光动画。
    
- 完美连锁：高饱和粉紫（#FF00D0），伴随粒子爆裂。
    
- 危险/障碍：警示橙红（#FF3B30）。

# 六、 项目里程碑与开发排期（MVP 14天验证计划）

小游戏讲究快速试错，建议分为以下三个阶段推进：

- 第 1–3 天：视觉与材质验证（美术先行）
    

- 目标：在 Blender 里通过代码或素材包拉出低模浮岛与水晶，导入 Cocos Creator。
    
- 产出：调出符合上述 5.2 规范的 Shader 材质，手动连一条发光线，验证“一秒吸睛”的视觉大方向。
    

- 第 4–8 天：核心玩法与模拟算法实现（程序介入）
    

- 目标：跑通等距网格系统与手牌拖拽逻辑，实现 GDD 4.3 的光路递归碰撞算法。
    
- 产出：玩家可在手机上拖拽卡牌、实时看到光路变化并顺利走到终点。
    

- 第 9–14 天：微信生态与商业化接入（裂变上线）
    

- 目标：接入微信群分享残局接口、体力充能接口以及腾讯广告 SDK（激励视频点位）。
    
- 产出：完成 20 关极简关卡，打包提审微信小游戏，进行首次种子用户测试。
